# Meu Site - Exemplo Simples com HTML

Este projeto é um exemplo básico de uma página HTML, utilizado para demonstrar conceitos iniciais de desenvolvimento web e também práticas com Git e versionamento de código.

## Objetivo

O objetivo deste projeto é:

 - Criar uma estrutura simples de página web
 - Trabalhar com elementos básicos do HTML
 - Simular o uso de branches no Git (ex: feature/criar_menu_navbar, feature/criar_section_aside, feature/criar_footer)
 - Servir como base para estudos e evolução do código
